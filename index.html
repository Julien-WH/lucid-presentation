Roadmap V2 : Tauri (Rust)

Empaquetage desktop natif — Binaire ultra-léger (<10Mo vs 150Mo Electron)

Stack technique de LUCID : choix délibérés pour la performance, la sécurité et la confidentialité. Landing page : AstroJS pour la génération de pages statiques ultra-rapides, Tailwind CSS pour le styling utilitaire, WordPress Headless comme CMS pour faciliter la gestion de contenu. Application frontend : React pour l'interface (composants réutilisables, écosystème riche), Vite pour le build (ultra-rapide), Tailwind CSS pour le styling cohérent avec la landing. Backend local : SurrealDB comme base de données graphe (relations complexes entre messages, personnes, événements), Ollama pour faire tourner des LLM en local (pas de cloud), Mistral comme modèle IA (open source, performant). Hébergement : Infomaniak, hébergeur suisse éthique et écologique. 100% énergies renouvelables, neutralité carbone, RGPD compliant, privacy by default. Roadmap V2 : Tauri pour l'empaquetage desktop. Pourquoi Tauri (Rust) plutôt qu'Electron ? Performance native (Rust compile en code machine), sécurité mémoire (pas de buffer overflow), binaire ultra-léger (moins de 10 Mo contre 150 Mo pour Electron). L'application web actuelle fonctionne déjà en local, Tauri apportera juste un wrapper natif pour une meilleure intégration système et des performances encore meilleures. Tout tourne en local : pas de serveur distant, pas de cloud, pas de fuite de données. C'est le cœur de la promesse Privacy by Design.
